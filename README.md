## ğŸ¯ Project Overview
The SkillPro web application is designed to support academic and administrative operations of an educational institute by providing:

- Online course information
- Student registration and enrollment
- Instructor management
- Course schedules and events
- Notices and inquiry handling
- Role-based access control

The system supports **three user roles**:
- **Admin**
- **Instructor**
- **Student**
